Instrucciones paso a paso Softgel

  1. Preparar uñas: Limpiar, empujar cutículas y limar.
  2. Limpiar superficie: Usar limpiador o alcohol para eliminar residuos.
  3. Aplicar las uñas : Elegir y preparar las uñas a colocar.
  4. Limpiar: Limpiar todo los residuos de las uñas.
  5. Esmaltar: Agregar la base coat con secado de 15 segundos y esmaltar.
  6. Aplicar top coat: Sellar y curar nuevamente.
  7. Eliminar capa pegajosa: Usar limpiador o alcohol.
  8. Hidratar: Aplicar aceite para cutículas.
